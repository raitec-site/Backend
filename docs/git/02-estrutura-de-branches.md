# 2. Explicação das branches

## main

Branch principal do projeto.

Representa a versão estável/produção.


## development

Branch principal de desenvolvimento.

Serve para integrar todas as funcionalidades antes de irem para a `main`.



## feature/*

Branches separadas por funcionalidade.

Cada parte do site possui sua própria branch.

Isso evita conflitos entre desenvolvedores.
