# 1. Padrão de commits

## feat

Nova funcionalidade.

```txt
feat: adiciona CRUD de membros
```



## fix

Correção de erro.

```txt
fix: corrige login admin
```



## style

Alterações visuais/CSS.

```txt
style: melhora layout do painel
```


## refactor

Refatoração de código.

```txt
refactor: reorganiza rotas admin
```


## docs

Documentação.

```txt
docs: adiciona manual git
```
